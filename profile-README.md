### Hi, I'm James

Genomics software engineer at the [Garvan Institute](https://www.garvan.org.au/), Sydney.
I build bioinformatics tools: file formats, signal processing, high performance analysis methods

Co-creator of [SLOW5/BLOW5](https://github.com/hasindu2008/slow5tools) (Nature Biotechnology), a high-performance alternative to ONT's FAST5/POD5 formats.
